## Installation

Install vue-lat1 dengan npm

```bash
  git clone https://github.com/ahmadsuryaalamsyah/Vue-lat1.git
  cd my-project
  npm i
```

## Deployment

Untuk menjalankan projek ini

```bash
  npm run dev
```
